# MERN Blog Platform

A full-stack MERN Blog Platform built using MongoDB, Express.js, React.js, and Node.js. The project includes user authentication, an admin dashboard, blog management, SEO optimization, comments, contact management, search, categories, and responsive design.

---

## Features

### User Panel
- View all blogs
- Search blogs
- Browse blogs by category
- Read blog details
- Comment on blogs
- Contact form
- Responsive design

### Admin Panel
- Secure Login
- Dashboard
- Create Blog
- Edit Blog
- Delete Blog
- Upload Blog Images
- Manage Comments
- Manage Contact Messages
- Settings Page
- Dark Mode

### SEO Features
- SEO Title
- Meta Description
- SEO-friendly Slug
- Canonical URL
- Open Graph Tags
- Twitter Card Tags

## Installation

### Clone Repository

```bash
git clone https://github.com/AlokeshG/MERN-Blog-Platform.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Admin Panel

```bash
cd admin-panel
npm install
npm run dev
```

---

## Environment Variables

Create a `.env` file inside the backend folder.

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```
